Bug Tracker

A simple MERN bug-tracker app with full test coverage examples and debugging exercises.
Users can report bugs, view, update status, and delete them. The project demonstrates unit & integration tests for backend and frontend, plus debugging techniques.

Features
Report new bugs
View list of bugs
Update bug status (open, in-progress, resolved)
Delete bugs
Unit & integration tests (Jest, Supertest, React Testing Library)
Error handling and Error Boundary on client
